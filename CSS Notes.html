Classes

You can class elements with < p class="brand">Sole Shoe Company< / p > this allows to style whatever is in that class.
then in css when styling it, it would look like this:

.brand {
}

You can also add more than one word for the class and chain elements. Example: h1.special

IDs

IDs are the most specific selector in CSS, followed by classes, and finally, tags

Styles and Boxes

Modify text with:

  • font-family
  • font-size
  • font-weight
  • text-align
  • color and background-color
  • opacity
  • background-image

Make a box with these Css styles:

  • height
  • width
  • padding
  • border
  • margin
  • overflow

Borders

Borders can be set with a specific width, style and color
There are 10 different styles and 140 built in different colors.
there are also the border-radius property
box-sizing: border-box; property takes into consideration the padding to make sure the height is exactly the number you want.

Padding and Other Properties

padding property is like line spacing
example: margin: 6px 12px;
margin: 0 auto; sets the top and bottom so the info is centered

The overflow property can be set to display, hide, or scroll, and dictates how HTML will render content that overflows its parent's content area. The visibility property can hide or show elements.
